
# نظام أورنيك (8) الرقمي - دليل النشر والتشغيل

تم تصميم هذا النظام ليكون حلاً وطنياً سريعاً وموثوقاً لإدارة الحوادث المرورية في السودان.

## متطلبات التشغيل
1. خادم ويب يدعم **HTTPS** (ضروري لعمل الـ PWA والموقع الجغرافي).
2. لا يتطلب قاعدة بيانات معقدة في المرحلة الحالية (يعتمد على LocalStorage و IndexedDB).

## خطوات النشر
1. ارفع كافة ملفات المشروع إلى المجلد الرئيسي في الخادم (public_html).
2. تأكد من أن ملف `sw.js` و `manifest.json` في الجذر (Root).
3. للوصول من أجهزة أندرويد: افتح المتصفح > أدخل الرابط > اختر "إضافة للشاشة الرئيسية".

## ميزات العمل الميداني
- **Offline-First**: يعمل التطبيق بالكامل بدون إنترنت.
- **Data Export**: يمكن للشرطي تصدير السجلات يدوياً كملف JSON من لوحة التحكم في حال عدم توفر مزامنة سحابية.
- **PDF Generation**: يتم توليد الأورنيك الرسمي محلياً داخل المتصفح وجاهز للطباعة أو الإرسال عبر واتساب.

## الأمن والخصوصية
- البيانات تُخزن مشفرة محلياً في ذاكرة المتصفح الآمنة.
- لا يتم إرسال أي بيانات لخوادم خارجية إلا عند ربط النظام بواجهة برمجة تطبيقات (API) رسمية تابعة للوزارة.
